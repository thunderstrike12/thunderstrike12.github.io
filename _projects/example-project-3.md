## Overview

Intuitive animation tool allowing designers to create stunning motion graphics directly in the browser with timeline editing and export capabilities.

## Key Features

- Keyframe-based timeline editor
- Layer management system
- Export to GIF, MP4, and WebM
- Template library for quick starts

## Technical Approach

Built with TypeScript for type safety, Canvas API for rendering, and GSAP for smooth animations. The architecture prioritizes performance for complex animations.
