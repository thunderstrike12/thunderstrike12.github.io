## Overview

Central hub application for managing IoT devices across different ecosystems. Features automation rules, energy monitoring, and voice control integration.

## Key Features

- Device discovery and management
- Automation rules engine
- Energy consumption tracking
- Voice assistant integration

## Technical Stack

Mobile app built with React Native, MQTT protocol for device communication, and Node.js backend for automation logic and data processing.
